# Build with Gradle
### Requirements
* java 11 or later

### How to build
```
./gradlew clean build
```
### How to run
```
cd build/libs
java -jar serving-web-content-0.0.1-SNAPSHOT.jar
```

# Build with Maven
### Requirements
* java 11 or later

### How to build
```
./mvnw clean package
```
### How to run
```
cd target
java -jar serving-web-content-0.0.1-SNAPSHOT.jar
```
